# Testing Checklist

## Gameplay Systems

### Player

- [ ] Movement functions correctly.
- [ ] Jumping behaves consistently.
- [ ] Collision detection works as expected.
- [ ] Player cannot leave playable bounds.

### Timeline System

- [ ] Timeline switching works instantly.
- [ ] HUD updates correctly.
- [ ] Visual effects trigger properly.
- [ ] Active objects update correctly.

### Puzzle Objects

- [ ] Platforms behave correctly.
- [ ] Doors unlock properly.
- [ ] Keys can be collected.
- [ ] Hazards trigger correctly.
- [ ] Exit portals function correctly.

---

## Level Validation

### Level 1

- [ ] Fully completable.
- [ ] Tutorial objectives clear.

### Level 2

- [ ] Puzzle logic functions correctly.
- [ ] Timeline interactions behave correctly.

### Level 3

- [ ] Puzzle solvable without exploits.
- [ ] Intended solution works reliably.

---

## User Experience

- [ ] Instructions are visible.
- [ ] Controls are understandable.
- [ ] Visual feedback is clear.
- [ ] Timeline differences are noticeable.

---

## Final QA

- [ ] Full playthrough completed.
- [ ] No soft-lock scenarios.
- [ ] No progression blockers.
- [ ] No game-breaking bugs.
