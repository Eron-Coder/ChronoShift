# AI Usage Report

## Overview

Artificial intelligence tools were used as development assistants throughout the creation of ChronoShift.

AI was not used to replace gameplay design or project ownership. Instead, it served as a productivity tool for ideation, debugging, documentation, and code refinement.

---

## Tools Used

- ChatGPT

---

## Areas of Assistance

### Development Support

AI assisted with:

- Debugging implementation issues.
- Reviewing game architecture.
- Refactoring code structures.
- Suggesting optimization strategies.

### Design Assistance

AI provided support for:

- Puzzle brainstorming.
- Timeline mechanic discussions.
- User experience improvements.
- Naming and thematic ideas.

### Documentation

AI assisted in drafting:

- README content.
- Architecture documentation.
- Technical documentation.
- Testing procedures.

---

## Human Contributions

The developer remained responsible for:

- System architecture.
- Gameplay implementation.
- Puzzle design.
- Visual design decisions.
- Feature prioritization.
- Testing and balancing.

---

## Statement

All AI-generated suggestions were reviewed, modified, and integrated manually by the project creator.
